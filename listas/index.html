Listas em HTML:

Existem 2 tipos de listas em html as ordenadas e as desordenadas:

Listas Ordenadas:

Sao listas em que a ordem dos itens importa:

  1. Item 1
  2. Item 2
  3. Item 3

Listar nao-Ordenadas:

Sao listas cuja ordem dos intes e irrelevante:

  • Item 1
  • Item 2
  • Item 3

Parametro Type:

Ambas as litas possuem o parametro type. Esse permite mudar a representacao dos itens:

  1. Letras Maiusculas
  1. Letras Minusculas
  1. Numeros Romanos com Letras Maiusculas
  1. Numeros Romanos com Letras Minusculas

  • Index Disco
  • Index Disco
  • Index Disco

Parametro Start:

Parametro exclusivo das listas ordenadas onde podemos decidir a partir de qual indice a lista comeca:

  1. Item 1
  2. Item 2
  3. Item 3

O start e um parametro numerico, portanto, mesmo que os seus indices sejam letras o start tem que ser passado como numero.

Listas Mistas:

Vamos colocar uma lista ordenada lentro de uma lista nao-ordenada:

  • Categorita 1:
    1. Item 1
